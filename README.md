# WGUPS Package Delivery System

## Overview

The WGUPS Package Delivery System is a Python application designed to efficiently manage and track the delivery of packages within the Salt Lake City area. The system utilizes a custom hash table and various algorithms to optimize package deliveries, ensuring timely and accurate distribution.

## Features

- **Efficient Package Management**: Utilizes a custom hash table to store package information, allowing for quick insertion, retrieval, and updating of package data.

- **Optimized Delivery Routes**: Implements a custom algorithm to optimize delivery routes, minimizing the total distance traveled by trucks and ensuring timely deliveries.

- **Real-time Package Tracking**: Provides real-time tracking of package statuses, including delivery times and locations, allowing for accurate monitoring of the delivery process.

- **Interactive User Interface**: Offers an intuitive user interface that allows users to query package statuses based on package IDs, delivery times, and specific time intervals.

## Getting Started

### Prerequisites

- Python 3.x
- CSV files containing address, distance, and package data

### Installation

1. Clone the repository to your local machine:

   ```bash
   git clone https://github.com/your-username/WGUPS-Package-Delivery.git
   ```

2. Navigate to the project directory:

   ```bash
   cd WGUPS-Package-Delivery
   ```

3. Run the main.py file to start the application:

   ```bash
   python main.py
   ```

## Usage

- Upon running the application, follow the prompts to input delivery times and query package statuses.
- Use the interactive interface to monitor package deliveries, track package statuses, and view total mileage traveled by trucks.
